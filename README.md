# 🍳 Foodie : Recipe Web App

Foodie is a full-stack web application that lets users search for any meal by name and instantly get full recipes  including ingredients, step-by-step instructions, and real calorie & macro data.



## ✨ Features

- 🔍 **Search any meal by name** — powered by TheMealDB API
- 📋 **Full recipe details** — ingredients and step-by-step instructions
- 🔥 **Real calorie & macro data** — protein, carbs, and fats per serving via Calorie Ninjas API
- 👤 **User authentication** — sign up, log in, or continue as guest via Supabase
- 🔖 **Save recipes** — logged-in users can save their favourite recipes
- 📱 **Fully responsive** — works on mobile and desktop

---

## 🛠 Tech Stack

### Frontend
| Tech | Purpose |
|------|---------|
| React + Vite | Frontend framework |
| Tailwind CSS | Styling |
| React Router | Page navigation |
| Supabase JS | Auth & database client |

### Backend
| Tech | Purpose |
|------|---------|
| Node.js + Express | REST API server |
| TheMealDB API | Recipe data (ingredients + instructions) |
| Calorie Ninjas API | Calorie & macro calculation |
| Supabase | User authentication & saved recipes database |



## 🌐 APIs Used

- [TheMealDB](https://www.themealdb.com/api.php) — free meal recipe API, no key required
- [Calorie Ninjas](https://calorieninjas.com/api) — nutrition data API, free tier available
- [Supabase](https://supabase.com) — open source Firebase alternative for auth and database
- [GoogleAIStudio]- Gemini AI Key
